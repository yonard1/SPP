# 🖥️ Aplikasi Sistem Informasi Pembayaran SPP

[![Laravel Version](https://img.shields.io/badge/Laravel-10.x-FF2D20?style=for-the-badge&logo=laravel&logoColor=white)](https://laravel.com)
[![Vite](https://img.shields.io/badge/Vite-5.x-646CFF?style=for-the-badge&logo=vite&logoColor=white)](https://vitejs.dev)
[![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-7952B3?style=for-the-badge&logo=bootstrap&logoColor=white)](https://getbootstrap.com)

Aplikasi berbasis web untuk pengelolaan dan monitoring transaksi pembayaran SPP siswa secara sistematis dan transparan.

---

## 🚀 Fitur Utama

Sistem ini mendukung **Multi-auth** dengan beberapa hak akses:
*   **Admin:** Manajemen data master (Siswa, Kelas, Petugas, Nilai SPP), input transaksi, dan cetak laporan.
*   **Petugas:** Input transaksi pembayaran dan melihat histori pembayaran.
*   **Siswa:** Login menggunakan NISN untuk mengecek histori dan status pembayaran pribadi.

---

## 🛠️ Teknologi & Library

*   **Framework:** Laravel
*   **Asset Bundler:** Vite
*   **Frontend UI:** Bootstrap
*   **Database:** MySQL / MariaDB
